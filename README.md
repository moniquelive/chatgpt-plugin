# Setup de ambiente

### Cria um venv:

```shell
python3 -mvenv venv
```

### Ativa

```shell
source ./venv/bin/activate
```

### Instala o poetry

```shell
pip install poetry
```

### Instala pacotes do poetry

```shell
poetry install
```

### Roda o exemplo:

```shell
python ./server/main.py
```

## OBS

Está com auto-reload, então pode mexer nos yamls e jsons que ele vai recarregar.
